Track logins (username + IP+COUNTRY+CITY).  Also, send nofitication to admin, when unknown user logins |OR| allow login only to specific IPs.


== Description ==

After plugin activation, go to  "Dashboard"->"Settings"->"Login Tracks". (SEE OTHER MUST_HAVE PLUGINS : <a href="http://codesphpjs.blogspot.com/2014/10/must-have-wordpress-plugins.html">NSP</a> )
 

== Installation ==

1. Download and extract archive to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
